About This Project

A personal geospatial photo explorer that visualizes my hiking and mountaineering trips across British Columbia. The app combines a high-performance photo gallery with an interactive map, allowing users to browse adventures either visually or geographically.

The interface features a responsive image grid with detailed trip pages and a full-screen lightbox viewer for immersive photo browsing. A synchronized map view plots every trip location using interactive markers, making it easy to explore hikes by region and quickly understand where each adventure took place.

The project showcases modern frontend and mapping technologies, including interactive web mapping with Mapbox, geospatial data visualization, and a dynamic gallery UI optimized for large photo collections. Images are organized with metadata (location, date, trip name) and rendered through a performant client-side architecture that keeps navigation fast even with hundreds of high-resolution photos.
